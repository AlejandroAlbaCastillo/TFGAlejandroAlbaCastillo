Introducción

Esta es una aplicación desarrollada para el proyecto final de Desarrollo de Aplicaciones Multiplataforma.

La idea general de la aplicación es que se pueda llevar un registro de las reservas de un complejo deportivo.

En esta aplicación se va a poder manejar:

  • Para el usuario

    • Poder crear una nueva reserva, así como modificarla o eliminarla

    • Registro de todas las reservas que ha hecho

    • Buscar una reserva especifica

  • Para el empleado (según sus permisos)

    • Llevar un registro de todo

    • Poder crear, modificar y eliminar cualquier elemento de la aplicación
